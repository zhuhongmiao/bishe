# 毕业设计 - 服装面料管理系统

一个基于 Vue + Python 的前后端分离毕业设计项目。

## 项目简介

本系统是一个服装面料管理平台，提供面料图片浏览、分类管理等功能。前端使用 Vue 3 + Ant Design Vue，后端使用 Python。

## 技术栈

### 前端 (FE)
- Vue 3
- Vite
- Ant Design Vue
- Pinia (状态管理)
- Vue Router

### 后端 (BE)
- Python
- Flask/FastAPI

## 项目结构

bishe/
├── BE/                 # 后端代码
│   ├── BE/            # 后端主目录
│   ├── venv/          # Python 虚拟环境
│   ├── .env           # 环境配置
│   ├── requirements.txt
│   └── 1.py           # 主程序
├── FE/                # 前端代码
│   ├── src/           # 源代码
│   ├── public/
│   │   └── fabrics/   # 面料图片资源
│   └── package.json
└── scripts/           # 脚本文件
## 快速开始
### 后端启动
```bash
cd BE
pip install -r requirements.txt
python 1.py
前端启动
